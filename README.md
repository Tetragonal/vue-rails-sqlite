## Development setup
    Install Ruby 2.4.4: https://cache.ruby-lang.org/pub/ruby/2.4/ruby-2.4.4.zip
    Install Yarn: https://yarnpkg.com/en/docs/install

    # Install dependencies
    bundle install
    yarn install
    
    # Run server (two separate consoles)
    ruby .\bin\webpack-dev-server
    rails s
